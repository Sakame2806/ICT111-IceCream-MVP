# MVP Experiment Plan

## 1. Group and Project Information
- Group name: Ice Cream
- Project title: ICT111-IceCream-MVP
- Repository link: https://github.com/Sakame2806/ICT111-IceCream-MVP
- Main target user: artists and Hobbyist
- Prototype platform: Backend/database prototype

## 2. Experiment Objective

To validate whether artists and Hobbyist can easily navigate IceCream to find artworks that match their preferred style, fandom, and critique needs.

## 3. Requirement Scope for the Experiment
List the requirement IDs that will be tested in this experiment.

| Requirement ID | Requirement Summary                       | Related Screen/Feature | Tested in This Experiment? |
|----------------|-------------------------------------------|------------------------|----------------------------|
| FR-07          | Detail view and Comments for each Artwork | Detail view            | Yes                        |
| FR-03          | User input/submission                     | Upload Page            | Yes                        |
| FR-06          | Search, filter, and Tagging function      | Tagging function | Yes |

## 4. MVP Experiment Type
Select one or combine more than one:
- [ ] Clickable prototype
- [ ] Landing page test
- [ ] Form-based simulation
- [ ] Dashboard demo
- [ ] Manual service simulation
- [x] Simple web prototype
