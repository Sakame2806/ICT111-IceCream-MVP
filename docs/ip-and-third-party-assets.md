## IP Summary
The IceCream prototype currently uses several third-party or external resources:

- **External libraries:** Tailwind CSS CDN and Font Awesome.
- **External image services:** Unsplash and Lorem Picsum.
- **Brand/design references:** Pixiv naming, logo text, premium labels, and Pixiv-style screen layouts.
- **Planned frameworks/APIs:** Vue.js 3 and Firebase Auth/Firestore.
- **Datasets:** Team-created mock records and validation results.
- **Possible AI-generated material:** No actual AI-generated artwork was found, but the upload form includes an AI-generated disclosure field.

The main IP concern is not the open-source libraries. The highest-risk items are the current Pixiv branding/design references, external placeholder images, and the unrelated `pimp-my-shoes.html` file. These should be replaced, removed, or clearly credited before the final prototype is submitted.
